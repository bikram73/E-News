# E-News 

A simple HTML-based news website displaying daily news, highlights, and weather information.

## Features

- **Breaking News** - Latest technology and education news
- **Today's Highlight** - Featured stories with formatted text (subscripts, superscripts, strikethrough)
- **Weather Report** - Current temperature for major Indian cities (Bangalore, Delhi, Mumbai)
- **Read More** - Link to BBC News for additional content

## Design

- Clean, card-based layout with white content boxes on a green background
- Responsive styling with Arial font family
- Color-coded sections (red for breaking news, green for highlights, orange for weather, purple for read more)

## Files

- `index.html` - Main news page

## How to Use

Open `index.html` in a web browser to view the daily news.

## Customization

You can easily customize:
- Background color in the `body` style
- News content in each section
- Weather data and cities
- External links
